Exploratory Data Analysis on Retail Sales Data using Python

## Project Objective
This project aims to demonstrate end-to-end Exploratory Data Analysis (EDA), starting from data cleaning to deriving insights through visualizations and statistical summaries to analyze Retail sales data using Python to identify sales trends,
top-performing products, regional performance etc..

## Dataset
- Domain: Sales / Retail
- Records: 500+
- Features: 10+ (Numerical & Categorical)

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis Performed
- Data cleaning and preprocessing
- Univariate, bivariate, and multivariate analysis
- Sales trend and category-wise analysis
- Region-wise and product-wise performance
- Correlation analysis

## Key Insights
- Certain product categories contribute significantly to total revenue
- Sales show seasonal patterns across months
- Regional differences impact overall performance
- High-selling products are consistent revenue drivers
- Discounts influence customer purchasing behavior

## Conclusion
The analysis provides actionable insights into retail sales performance and demonstrates
the importance of data-driven decision-making in real-world business scenarios.
